kinisi-agent-nodejs
===================

Kinisi Agent written in NodeJS


### Getting Started

Start on local mocks (without a GPS device or remote kinisi server):

```sh
$ npm install
$ export NODE_ENV=mock
$ node kinisi-agent.js
```
